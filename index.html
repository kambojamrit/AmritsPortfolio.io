Academic Exchange Platform (AEP)

The Academic Exchange Platform is a Java based web application that enables academic professionals and institutions to connect for educational collaboration and teaching opportunities. Academic Institutions can post course offerings, while Academic Professionals can search, view, and request to teach these courses based on expertise.

Smart Image Download Organizer

Have you ever downloaded a file, only to lose it in a messy downloads folder? I built a Smart Download Organizer for my image downloads, a lightweight PowerShell script that automatically sorts new downloads into designated folder based on file type. No more manual dragging and dropping, just seamless organization in real time.
